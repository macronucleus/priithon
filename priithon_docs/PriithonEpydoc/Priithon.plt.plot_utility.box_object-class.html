Class box_object

source code

Helpful for laying out rectangles.

This class has three general areas of functionality:

  1. Retreive information about a box such as it's size, the x-coordinate of its left side, and others.
  2. Specify the location of one box (rectangle) in relation to another using methods such as left_of(), above(), or center_on().
  3. Change the size (and location) of a box by either choping off a portion of the box using trim_right(), trim_left(), and others or inflating the box by a percentage.
