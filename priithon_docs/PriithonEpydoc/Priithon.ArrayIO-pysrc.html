 11  """This module contains elementary support for I/O of one- and 
 12  two-dimensional numerical arrays to and from plain text files. The 
 13  text file format is very simple and used by many other programs as 
 14  well: 
 15   
 16  - each line corresponds to one row of the array 
 17   
 18  - the numbers within a line are separated by white space 
 19   
 20  - lines starting with # are ignored (comment lines) 
 21   
 22  An array containing only one line or one column is returned as a 
 23  one-dimensional array on reading. One-dimensional arrays are written 
 24  as one item per line. 
 25   
 26  Numbers in files to be read must conform to Python/C syntax.  For 
 27  reading files containing Fortran-style double-precision numbers 
 28  (exponent prefixed by D), use the module Scientific.IO.FortranFormat. 
 29  """
