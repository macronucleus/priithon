SetXSpec(self, type='auto')

source code  
xSpec- defines x axis type. Can be 'none', 'min' or 'auto'
where:
    'none' - shows no axis or tick mark values
    'min' - shows min bounding box values
    'auto' - rounds axis range to sensible values
