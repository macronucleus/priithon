Read in an SPE file and output a FITS file

Example usage: Given an SPE file named test.SPE which contains 3 image frames, then the following code will produce three FITS images, named: . image_x001.fits . image_x002.fits . image_x003.fits >>> import piUtils >>> piUtils.speToFits('test.SPE', fitsfile='image')

readSpe(spefilename, verbose=False, onlyHdr=False)

source code  
Read a binary PI SPE file into a python dictionary

Inputs
spefilename --  string specifying the name of the SPE file to be read
verbose     --  boolean print debug statements (True) or not (False)
onlyHdr     --  if True, only read header, and skip filling `data` with image data

Outputs
spedict     --  python dictionary containing header and data information
                from the SPE file
                Content of the dictionary is:
                spedict = {'data':[],    # a list of 2D numpy arrays, one per image
                'IGAIN':pimaxGain,
                'EXPOSURE':exp_sec,
                'SPEFNAME':spefilename,
                'OBSDATE':date,
                'CHIPTEMP':detectorTemperature
           }

Example usage:
Given an SPE file named test.SPE, you can read the SPE data into
a python dictionary named spedict with the following:
>>> import piUtils
>>> spedict = piUtils.readSpe('test.SPE')

speDictToFits(spedict, outfile=None, clobber=False, verbose=False)

source code 

Given a python dictionary containing information about the SPE file generate a sequence of FITS files, one FITS file per image in the SPE file.

Inputs spedict -- A dictionary containing information from the SPE file . as output by readSpe() outfile -- root of output filename for FITS files. . e.g. outfile='fitsimage' will produce fits files . named fitsimage_x000.fits clobber -- Boolean. True will overwrite any existing FITS files. verbose -- Print debug messages.

Outputs fitsFileNames -- Returns a list of the generated output filenames.

Example usage: If the file called test.SPE contains 3 image frames, then the following code: >>> import piUtils >>> spedict = piUtils.readSpe('test.SPE') >>> piUtils.speDictToFits(spedict, outfile='image') will produce three FITS images, named: . image_x001.fits . image_x002.fits . image_x003.fits
