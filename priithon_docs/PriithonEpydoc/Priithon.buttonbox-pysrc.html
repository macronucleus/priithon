  1  """ 
  2  A buttonBox is a list (maybe many rows) of buttons, labels, txtCtrls, checkBoxes 
  3   
  4  Create a new buttonBox with  
  5  >>> buttonBox(...) 
  6  it gets automatically appended to the list of all `buttonBoxes`. 
  7   
  8  You can (dynamically) add one or more controls with `buttonBoxAdd()`. 
  9  Both `buttonBoxAdd()` and `buttonBox()` take the same `itemList` argument 
 10     each `item` in `itemList` is translated into a call of 
 11       1) startNewRow(weight, expand) 
 12     or 
 13       2) addButton(label, cmd, weight, expand, tooltip, iniFcn) 
 14   
 15  if `item` is a list or tuple, up to 4 things can be specified: 
 16       1.) `label` (i.e. text on button or text in textCtrl) 
 17       2.) `command`  (default: None) 
 18       3.) `weight` - items will get screen-space by relative weight (default: 1) 
 19       4.) `expand` - adjust size vertically (on resizing buttonBox) (default: True) 
 20       5.) `tooltip` - set mouse-over tooltip (default: the cmd string) 
 21    .if `item` is a string, it specifies `label` and uses the above defaults for all others 
 22   
 23  `label` has the one of 3 forms (note the SPACE and the TAB): 
 24          "C"  ,  "A\tC"  or   "A B\tC" 
 25    "A" specifies the wx-control type: 'b','tb','t','c','l','sl' 
 26        default is 'b' 
 27        'b'  button 
 28        'tb' togglebutton 
 29        't'  text-control 
 30        'c'  checkbox 
 31        'r'  radiobutton 
 32        'l'  static-label 
 33        'sl' slider 
 34   
 35    "B" specifies `exec_to_name_control` - a string that is executed at creation time,  
 36        exec in execModule - use 'x' to refer to the wx-control, '_' to the execModule 
 37        default is '' 
 38    "C" specifies the wx-control"s label 
 39   
 40    note 0) label without '\t' defaults to control-type button (A => 'b') 
 41    note 1) for button: if cmd is None: use label (part C) as cmd 
 42    note 2) for txtCtrl: if cmd is '--': make read-only, set cmd to None 
 43    note 3) for checkbox: if label (part C) contains (another) '\t' use ALIGN_RIGHT 
 44    note 4) for radiobutton: if label (part C) contains (another) '\t' use RB_GROUP (to start a new group) 
 45    note 5) for label: command is ignored #CHECK 
 46    note 6) for slider: label gives value,minValue,maxValue (space separated) - default: 0,0,100 
 47   
 48   
 49  `cmd` (`command`) will get exec'ed when the button is pressed, or the text is changed 
 50    if will get exec'ed with globals been the buttonbox's execModule 
 51                         and locals as follows: 
 52     `x`: the control's value 
 53     `_`: the execModule  -- so that you can write "_.x = x", instead of "globals()[x] = x" 
 54     `_ev`: the wx-event (wxButtonEvent, ....) 
 55     `_b`: the wxControl (the buttonbox's "button", or now more general, it's "gui-control") object 
 56           ( same as _ev.GetEventObject() )
